# uv(Conda보다 압도적으로 빠름) rust로 만듬
- https://github.com/astral-sh/uv

# install

- 1. Rust 를 Install한다.

- 2. Rust 를 이용한 Install

```
cargo install --git https://github.com/astral-sh/uv uv  
```

- 3. Path확인

- `.cargo/bin` 에 잘 설치 되었는지 확인 `.cargo/bin`이 path가 잘 되었는지 확인

```
echo $PATH  
```

- 4. update

```
uv self update  
```
